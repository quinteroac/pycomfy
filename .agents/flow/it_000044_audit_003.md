## Minor Observations

1. `cli/__main__.py` docstring says `"Allow python -m cli invocation"` — should say `python -m parallax` for clarity, since the canonical entry is `parallax/__main__.py`.
2. **FR-3 gap**: `PARALLAX_UV_PATH` env var is not read in `cli/_async.py._uv_path()`; `shutil.which("uv")` is used instead. `PARALLAX_RUNTIME_DIR` is also not read anywhere in the Python CLI.
3. `cli/commands/upscale.py` uses `input: str = ...` (bare Ellipsis) to mark `--input` as required — unconventional for Typer; consider `typer.Option(..., help="...")`.
4. `save_video_frames()` PyAV fallback writes PNG frames to a directory and returns a directory path rather than an `.mp4` path — callers should handle both output types.
5. No integration tests cover actual pipeline calls in sync mode (pipelines are mocked); this is acceptable given the CPU-only CI constraint documented in `PROJECT_CONTEXT.md`.

---

## Conclusions and Recommendations

The CLI prototype is production-ready for all four user stories. The only actionable compliance gap is **FR-3**:

- `PARALLAX_UV_PATH` should be read in `cli/_async.py._uv_path()` before falling back to `shutil.which("uv")`.
- `PARALLAX_RUNTIME_DIR` should be read in `cli/_async.run_async()` to override `script_base` (currently hardcoded to `_REPO_ROOT`).

All other observations are cosmetic or low-priority.

## Refactor Plan

### P1 — FR-3: Honour `PARALLAX_UV_PATH` env var (high priority)

**File:** `cli/_async.py`

```python
def _uv_path() -> str:
    import os
    env_path = os.environ.get("PARALLAX_UV_PATH")
    if env_path:
        return env_path
    found = shutil.which("uv")
    return found if found else sys.executable
```

### P2 — FR-3: Honour `PARALLAX_RUNTIME_DIR` env var (high priority)

**File:** `cli/_async.py`

```python
import os as _os
_REPO_ROOT = _os.environ.get("PARALLAX_RUNTIME_DIR") or str(Path(__file__).resolve().parents[1])
```

Or make it dynamic in `run_async()`:

```python
script_base = os.environ.get("PARALLAX_RUNTIME_DIR") or _REPO_ROOT
data = JobData(..., script_base=script_base, ...)
```

### P3 — Cosmetic: Fix `cli/__main__.py` docstring (low priority)

Change `"Allow ``python -m cli`` invocation."` → `"Allow ``python -m parallax`` invocation."`.

### P4 — Style note: `upscale.py` required `--input` (not actionable)

`= ...` (Ellipsis) is the correct Typer idiom for a required `Option` when it follows a parameter with a default (removing the default would cause a `SyntaxError`). No change needed.
