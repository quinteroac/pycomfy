## Minor Observations

1. `--cfg`, `--bpm`, and `--lyrics` are always appended to the subprocess args (even when using their defaults). This is benign because the Python script's defaults match the CLI defaults, but it differs slightly from the optional-flag pattern used for `--seed`.
2. `tsc --noEmit` reports pre-existing type incompatibilities between `bun-types@1.3.11` and the global `lib.dom.d.ts` (`WebAssembly.ImportExportKind`). These errors are unrelated to this iteration's code and were present before PRD 004 implementation.
3. The `create audio` command's `--length` option description says "Duration in seconds" which accurately maps to the script's `--duration`; noting the rename in help text would further clarify the forwarding for users.
